# 🌿 EcoTrace — Carbon Footprint Awareness Platform

> Built for **PromptWars Virtual – Challenge 3** using **Gemini AI** and **Google Antigravity**

EcoTrace helps individuals understand, track, and reduce their personal carbon footprint through a detailed calculator, personalized AI insights, daily habit tracking, and goal setting.

## ✨ Features

| Feature | Description |
|---|---|
| **Carbon Calculator** | 4-category calculator (Transport, Home, Food, Shopping) with India-specific emission factors from CEA 2024 & IPCC AR6 |
| **Personal Dashboard** | Circular emission gauge, comparison chart vs India & global averages |
| **Gemini AI Insights** | Personalized 5-action reduction plan generated by Gemini 2.0 Flash |
| **EcoAI Chatbot** | Real-time streaming chat with Gemini for sustainability Q&A |
| **Daily Habit Tracker** | 12 eco-habits with streak counter and achievement badges |
| **Goal Setting** | Set a CO₂ reduction target and track progress |
| **Firebase Auth** | Google Sign-in + Email/Password authentication |

## ⚙️ Setup Instructions

### Step 1 — Prerequisites
Install **Node.js** (v18+) from https://nodejs.org (choose LTS)

Verify: open a terminal and run:
```bash
node -v    # should print v18+ or v20+
npm -v     # should print 9+
```

---

### Step 2 — Get your Gemini API Key
1. Go to https://aistudio.google.com/apikey
2. Click **Create API key**
3. Copy the key — you'll need it in Step 4

---

### Step 3 — Create a Firebase Project
1. Go to https://console.firebase.google.com
2. Click **Add project** → name it `ecotrace` → create
3. In the left sidebar click **Authentication** → **Get started** → enable **Email/Password** and **Google**
4. In the left sidebar click **Firestore Database** → **Create database** → choose **Start in test mode** → pick a region
5. Click the **⚙️ gear icon** → **Project settings** → scroll to **Your apps** → click **</>** (Web)
6. Register the app (name: `ecotrace-web`) — copy the `firebaseConfig` object values

---

### Step 4 — Configure environment variables
In the project root, copy the example file:
```bash
cp .env.example .env
```
Open `.env` and fill in your values:
```
VITE_GEMINI_API_KEY=your_gemini_key_here

VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your-project-id
VITE_FIREBASE_STORAGE_BUCKET=your-project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
```

---

### Step 5 — Install dependencies & run locally
```bash
npm install
npm run dev
```
Open http://localhost:5173 in your browser.

---

### Step 6 — Deploy to Firebase Hosting (live URL for judges)
```bash
# Install Firebase CLI (one-time)
npm install -g firebase-tools

# Login
firebase login

# Initialize hosting in your project folder
firebase init hosting
# When asked:
#   - Use existing project → select your Firebase project
#   - Public directory → dist
#   - Single-page app → Yes
#   - Overwrite index.html → No

# Build the production bundle
npm run build

# Deploy
firebase deploy --only hosting
```
Your app will be live at: `https://your-project.web.app`

## 🤖 How Gemini AI is Used

1. **Personalized Insights**: User's emission data is sent to `gemini-2.0-flash` with a structured prompt. Gemini returns 5 specific, India-relevant recommendations in JSON format, including impact estimates and difficulty ratings.

2. **Streaming Chatbot**: Users can ask sustainability questions via an inline chat. Responses stream word-by-word using `sendMessageStream()` — creating a real-time typing effect.

### Example Gemini Prompt (Insights)
```
You are EcoAI, a carbon footprint expert focused on India.
User: Transport 1,820 kg | Home 850 kg | Food 420 kg | Shopping 200 kg
Top source: Transport
→ Generate 5 specific, India-relevant actions in JSON format
```
